# SEO-Optimized Article Writer Skill

一个完全自动化的 SEO 文章写作系统，为 Claude 设计。

## 功能

- 自动 Google 搜索排名前 5 的文章
- 智能文章内容提取
- 竞争对手字数分析
- SEO 优化的大纲生成
- AI 写作提示生成（包含 AI 去重指令）
- SEO 标题和 Meta Description 建议
