# SEO Article Extractor 的 URLs 示例文件
# 
# 使用说明：
# 1. 在 Google 上搜索你的关键词
# 2. 找出排名前 5 的高质量文章
# 3. 确保每个链接都能正常打开
# 4. 将 URL 复制到这个文件中（每行一个）
# 5. 将文件重命名为 urls.txt
# 6. 运行脚本：python3 seo_article_extractor.py "你的关键词"
#
# 示例：关键词 "What Is LED & How It Works"
